Before Making the Project Live:

  • Make the database using counsellingmait.sql file
  • In settings.py file, change the ALLOWED_HOSTS
  • In settings.py file, change the DEBUG to False
  • In settings.py file, change the 'NAME', 'USER', 'PASSWORD', 'HOST', and 'PORT' in DATABASES
  • In settings.py file, change the 'EMAIL_HOST_PASSWORD'
  • In templates/counselling.html file, change the fees link
  • In templates/btech/btech7.html file, change the fees link (Do the same for all other courses)
  • In the allowed_ip_addresses table in the database, add '0.0.0.0' (without quotes) if you want to allow everyone to register and login
  • In the allowed_ip_addresses table in the database, add the specified IP addresses if you want to allow only those to register and login

After Making the Project Live, Create a Superuser by the Following Command:

python manage.py createsuperuser

How to Use the Admin Panel:

  • Go to http://hostname.mait.ac.in:8000/admin/ and login with the superuser credentials
  • To generate PDF of a submitted form, select the record and go to action -> 'Generate PDF file' -> Go -> Print PDF for xyz rank
  • By default, no student is allowed to edit their record. In order to allow a student to edit his/her record, simply select the checkbox and click on Save
  • By default, no student is allowed for Counselling fees submission. In order to allow a student for counseling, simply select the checkbox and click on Save
  • You can also allow/disallow student(s) for editing and counseling by selecting their record and action -> allow/disallow -> Go
  • You can Filter students by various criteria. You will find this option on the right side of the page
  • You can Sort on the basis of any field by just clicking on the field name
  • You can Search any record by their Name or IPU Registration Number. You will find the Search option on top of the page
  • You can export any table's records in xls, csv, etc. formats. To do so, select the records -> action -> export -> format -> select format -> go
